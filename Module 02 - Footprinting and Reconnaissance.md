Tools Used :  Windows 10 virtual machine, Parrot Security virtual machine, Web browsers with an Internet connection, Administrator privileges to run the tools

The objective of this Module is to extract information about the target organization that includes Organization Information, Employee details, partner details, weblinks, web technologies, patents, trademarks, etc.
Network Information such as Domains, sub-domains, network blocks, network topologies, trusted routers, firewalls, IP addresses of the reachable systems, the Whois record, DNS records, and other related information
and System Information such as Operating systems, web server OSes, user accounts and passwords, etc
